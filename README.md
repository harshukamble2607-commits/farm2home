# farm2home

Farm to Home is a streamlined e-commerce solution designed to connect farmers directly with consumers. This project leverages the creative power of Canva AI for its interface and the flexibility of Google Sheets as a powerful, real-time backend and database.
​
The Tech Stack
​Frontend UI: Designed and powered by Canva AI for a modern, responsive user experience.
​Backend & Database: Google Sheets (serves as the centralized data hub for products, users, and orders).
​Automation: [Google Apps Script] to sync Canva actions with the Spreadsheet.
​
Project Interfaces
​1.User Interface
​Browse live product catalogs synced directly from Google Sheets.
​Place orders via integrated forms.
​View fresh produce updates generated by Canva AI visuals.

2.​ Farmer Dashboard-
​Dedicated portal to input crop data.
​Update stock levels which instantly reflect on the main website.
​View specific order requests assigned to their farm.

​3. Admin Control Center (Full Authority)
​The Admin manages the entire platform directly through the Google Sheets Backend:
​Inventory Control: Modify, add, or delete any product row.
​User Management: Approve or block Farmer and User accounts.
​Global Updates: Any change made in the Sheet automatically updates the Canva-powered frontend.
​Order Oversight: Full visibility into every transaction across the platform.

​ Database Structure (Google Sheets) :-
​The backend is organized into the following key sheets:
​Products: Name, Price, Farmer ID, Stock Level, Image URL.
​Users: Name, Role (Farmer/User/Admin), Contact Info.
​Orders: Order ID, Product Link, Status, Timestamp.

​ How to Set Up:-
​Canva Template: Access the Canva project link and ensure the AI modules are connected to your data source.
​Google Sheet Setup: * Create a copy of the provided Google Sheet template.
​Enable "Share" permissions to allow the frontend to read/write data.
​API Connection: Connect the Sheet ID to your Canva UI using your preferred integration tool.
